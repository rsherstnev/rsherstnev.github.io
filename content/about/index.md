[Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) — это тема для блога, изначально основанная на [Jekyll](https://jekyllrb.com/). Из-за ограничений дизайна Jekyll нативно не поддерживает интернационализацию (i18n) и требует сторонних плагинов для этой функциональности. Чтобы реализовать поддержку i18n для Chirpy без необходимости полагаться на сторонние плагины, проект [hugo-theme-chirpy](https://github.com/geekifan/hugo-theme-chirpy) перенёс тему Chirpy на [Hugo](https://gohugo.io/) с минимальными адаптациями. Все возможности Chirpy доступны в hugo-theme-chirpy (хотя некоторые функции могут работать иначе в рамках Hugo).

Следуйте постам на этом сайте, чтобы быстро создать бесплатный личный блог!

## Возможности

- **Тёмный режим**: повышенная читаемость при слабом освещении.
- **Эффективная организация записей**: иерархические категории, популярные теги, рекомендуемые материалы и поиск.
- **Оптимизированный макет**: содержание (TOC), подсветка синтаксиса, подсказки и многое другое.
- **Богатые расширения для написания**: поддержка математических формул, диаграмм, блок-схем и встроенных медиа.
- **Несколько систем комментариев**: выбор из различных вариантов для комментирования.
- **Инструменты веб-аналитики**: интеграция с несколькими аналитическими сервисами.
- **Современные веб-технологии**: создан для SEO и производительности.
- **Поддержка RSS-лент**: держите читателей в курсе событий через RSS.
